# Cheat Sheet

![xcode-cheat-sheet](https://user-images.githubusercontent.com/86648892/210223520-58a21db4-ce20-43f3-9f7a-8cab3db82464.png)

- cmd + shift + L
  - Library 메뉴
- cmd + shift + O
  - 빠른 검색(Open Quickly)
- cmd + shift + left, right, up, down
  - 해당 방향 커서 나머지 부분 전부 선택
- cmd + shift + F
  - 레퍼런스 검색
- cmd + shift + J
  - 현재 파일 디렉토리 위치로 이동
- option + left, right
  - 단어 단위로 이동
- cmd + left, right
  - 라인 끝으로 이동
- ctrl + shift + up, down
  - 커서 여러 개 생성
- cmd + b
  - Build
- cmd + r
  - Build and Run
- cmd + /
  - 해당 라인 코드 comment화 on / off
- cmd + [
  - 해당 라인 코드 왼쪽으로 이동
- cmd + ]
  - 해당 라인 코드 오른쪽으로 이동
- cmd + shift + option + enter
  - source control
